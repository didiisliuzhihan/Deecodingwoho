🧠Reflection:

Coding is no longer just a language for programmers, more designers, artists or other industries are also involved. When I first came into contact with creative coding, I chose this major with the intention that artists also need to be exposed to the language tools of the new era.

Coding is a way of expression, just like painting, photography, games, writing and other creative languages. It is also based on the Internet, which allows people to have more diverse access to code works from different brains, so that they can communicate and understand each other. This is a process that is a bit difficult to get started, but it is very interesting.

Now I think coding is like architecture. It is the bricks and foundation of the Internet world. It also depends on the programmers themselves. Do you want to build a playground for people to have fun, an office building that provides a production platform, or a library for communication and learning? Or a diverse community that includes everything? Everything is possible.
